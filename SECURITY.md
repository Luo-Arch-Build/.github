# Security Policy

## Reporting a vulnerability

Email: luoarch@proton.me

Please include:
- A description of the issue and its impact
- Steps to reproduce, proof of concept, or affected commit/version
- Any suggested remediation

## Response targets

- **Acknowledgement:** within 3 business days
- **Initial assessment:** within 7 business days
- **Coordinated disclosure:** target 90 days from first contact, sooner for actively exploited issues

Please do not file public issues for suspected vulnerabilities. Reporters will be credited in release notes unless anonymity is requested.
